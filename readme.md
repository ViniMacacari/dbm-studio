# DBM Studio

**DBM Studio (Database Management Studio)** is a modern EA FC database editor and modding tool built with Electron, Angular and TypeScript.

Created by **Vinícius Macacari**, DBM Studio was inspired by **Rinaldo's DB Master**, one of the most iconic database editing tools in the FIFA modding community. The goal of this project is to bring the same flexibility of traditional table editing while introducing modern visual editors and workflow-oriented tools for EA FC modders.

## Features

### Database Support

* Open and edit EA FC `.db` files.
* Open and edit EA FC `.xml` files.
* Save supported database formats.
* Fast database loading and navigation.
* Search, sort and filter records.

### Table Editor

* Spreadsheet-style editing inspired by DB Master.
* Direct access to database tables.
* Copy, paste, replace and delete records.
* Import and export tables.
* Bulk editing support.

### Visual Editors

Unlike traditional database tools, DBM Studio provides dedicated visual interfaces for common editing tasks.

#### Players

* Create new players.
* Edit attributes and personal information.
* Manage player identities and names.

#### Teams

* Create new clubs.
* Edit team information.
* Manage team-related data.

#### Leagues

* Create new leagues and competitions.
* Edit league settings and properties.

### Transfers

* Transfer players between clubs quickly.
* Simplify roster management.
* Reduce manual editing across multiple tables.

### Utilities

* Hash ID generation.
* XML descriptor parsing.
* Database inspection tools.
* BIGF/BIG4 archive extraction.
* Raw FIFA database reading support.

## Why DBM Studio?

DB Master revolutionized FIFA database editing by giving modders direct access to game databases.

DBM Studio builds upon that idea by combining traditional spreadsheet editing with modern visual tools, making database editing faster, more accessible and more efficient for today's EA FC modding community.
